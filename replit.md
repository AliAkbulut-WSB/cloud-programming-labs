# Cloud Programming Labs

## Overview
This is a JavaScript programming labs template project. It provides a web interface showing available programming tasks for cloud computing exercises.

## Project Structure
- `index.js` - Main server file serving the web interface on port 5000
- `js/` - Contains task directories (task0, task1, task1b, task1c, task2, task2b, task3a, task3b, task3c) for lab assignments

## Tech Stack
- Node.js 20
- Native HTTP server (no external dependencies)

## Running the Project
The server runs on port 5000 and is accessible via the web preview.
